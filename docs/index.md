# JsonPatchX

A PATCH framework for Python.

JsonPatchX extends RFC 6902 when an API needs more than a generic patch
document.

!!! note "Alpha status" JsonPatchX is in alpha. Expect iteration, and report
anything that feels unclear, rough, or broken.

JsonPatchX gives you a path from plain RFC 6902 patching to governed PATCH APIs
with validation, clearer OpenAPI, route-level policy, and explicit extension
points.
