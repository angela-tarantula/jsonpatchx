# JsonPatchX

JsonPatchX is a PATCH framework for Python, from standard
[RFC 6902](https://datatracker.ietf.org/doc/html/rfc6902) to governed PATCH APIs
and agent-friendly patch contracts.

Start with the [User Guide](user-guide/getting-started.md). It begins with plain
RFC 6902 patching and builds from there.

Read [About](about/about.md) for why the project exists and where it is trying
to push PATCH API design next.
